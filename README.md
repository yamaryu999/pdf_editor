# PDF Editor

小規模チームでも扱いやすい Python 製の簡易 PDF 編集アプリです。PySide6 を使ったデスクトップ UI を備えており、既存 PDF の閲覧・ページ選択・画像の挿入と調整・PDF への保存が可能です。

## 概要

- 既存の PDF を読み込み、ページごとにプレビュー表示
- 任意のページに画像（PNG/JPEG 等）を挿入し、位置・サイズを調整
- 編集結果を新しい PDF ファイルとして書き出し

## 必要環境

- Python 3.11 以上
- Linux / macOS / Windows いずれか（グラフィカル環境が必要）

## セットアップ

```bash
cd pdf_editor
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -e .
```

## 起動方法

上記のセットアップが完了している前提です。

```bash
# 仮想環境を有効化した上で
pdf-editor
```

もし `pdf-editor` コマンドが見つからない場合は、モジュールとして起動できます。

```bash
python -m pdf_editor.app
```

## 画面構成

- 左ペイン：ページ一覧（サムネイル付きのリスト）
- 中央：選択中ページのプレビュー＆編集キャンバス
- 右ペイン（ドック）：選択した画像の座標やサイズを編集するプロパティパネル
- 上部メニュー／ツールバー：
  - ファイル > 開く … PDF ファイルを読み込み
  - ファイル > 名前を付けて保存 … 編集後 PDF を書き出し
  - 編集 > 画像挿入 … 画像ファイルを挿入

## 基本的な使い方

1. 「ファイル > 開く」から PDF を選択  
2. 左側のページ一覧で編集したいページをクリック  
3. メニューまたはツールバーの「画像挿入」ボタンから画像ファイルを選択  
4. 挿入された画像をキャンバス上でドラッグして位置を調整  
5. より細かい調整が必要な場合は、右側プロパティパネルの X / Y / Width / Height を変更  
6. 「ファイル > 名前を付けて保存」で編集済み PDF を書き出し
